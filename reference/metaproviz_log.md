# Browse the current MetaProViz log file

Browse the current MetaProViz log file

## Usage

``` r
metaproviz_log()
```

## Value

Returns `NULL`.

## See also

[`metaproviz_logfile`](metaproviz_logfile.md)

## Examples

``` r
if (FALSE) { # \dontrun{
metaproviz_log()
# then you can browse the log file, and exit with `q`
} # }
```
