# Path to the current MetaProViz log file

Path to the current MetaProViz log file

## Usage

``` r
metaproviz_logfile()
```

## Value

Character: path to the current logfile, or `NULL` if no logfile is
available.

## See also

[`metaproviz_log`](metaproviz_log.md)

## Examples

``` r
metaproviz_logfile()
#> [1] "/__w/MetaProViz/MetaProViz/docs/reference/metaproviz-log/metaproviz-20251120-1331.log"
# [1] "path/metaproviz/metaproviz-log/metaproviz-20210309-1642.log"
```
